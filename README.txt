# 🎲 Dice Roller

A simple dice rolling application built using Python and Tkinter. Click the button to roll the dice with a smooth animation and get a random result.

## ✨ Features

* 🎲 Random dice roll (1–6)
* 🔄 Smooth rolling animation
* 🖥️ Simple and clean GUI

## 🛠️ Requirements

* Python 3.x
* Tkinter (comes pre-installed with Python)

## 🚀 How to Run

1. Clone the repository:

```
git clone https://github.com/your-username/dice-roller.git
```

2. Navigate to the project folder:

```
cd dice-roller
```

3. Run the program:

```
python dice_roller.py
```
